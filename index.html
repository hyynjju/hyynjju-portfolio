고령자나 시각 장애인을 포함한 디지털 소외 계층도 쉽게 사용할 수 있는 복약 관리 앱의 UI/UX 설계 및 프론트엔드 개발을 담당했습니다.

기존 복약 앱은 복잡한 조작으로 인해 사용자 이탈과 복약 누락, 중복 복용 등의 문제가 있었습니다. 이를 해결하기 위해 사용자가 직접 조작하지 않아도 자동으로 복약이 기록되는 UX를 설계하고, 음성 입력과 NFC 태그 기반의 비접촉 기능을 직접 구현했습니다.

또한, 95% 이상의 정확도를 가진 알약 이미지 인식 기능을 통해 약 이름이나 패키지를 식별하기 어려운 사용자도 쉽게 의약품 정보를 확인할 수 있도록 구성했습니다.

기술적으로는 React Native를 활용해 음성, 이미지, NFC 기반 로직을 포함한 UI 인터랙션을 직접 구현하고, 재사용성과 유지보수성을 고려한 컴포넌트 구조로 개발에 기여했습니다.

이 경험을 통해 누구도 소외되지 않는 사용자 경험을 고민하게 되었으며, 앞으로도 사회적 가치를 만드는 프로덕트를 만들고 싶습니다.
